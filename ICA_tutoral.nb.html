ICAnet tutorial

Weixu wang

2020年7月2日

ICAnet can run on both single batch and multi-batch datasets. For each datasets, we run ICA independently, further, we cluster the all independent components, find shared and specific expression programs, and combine with PPI to perform clustering. In this tutorial, I gonna used cell line dataset provided by 10X Genomics as illustate dataset to show how to use ICAnet to perform batch effect correction and clustering. The source dataset(including gene expression dataset, cell annotation, and PPI network) can be download from https://github.com/WWXkenmo/MouseGerm. We load the dataset at the first,

PPI <- readRDS("PPI_for_cell_line.RDS")

ID <- read.table("ID_for_cell_line.txt",header=TRUE,row.names=1,stringsAsFactors = FALSE)
